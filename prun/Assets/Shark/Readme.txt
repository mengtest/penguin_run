Description:

This giant flesh eating fish model has been designed to work well in video games and other interactive projects.  The model comes with several animations along with a workable script to achieve natural looking swimming motions with varying speeds.

How to use:

Simply drag the shark.fbx file in the hierarchy and you are good to go.

Using Control:

To use controls, simply assign the SharkMovement script to the shark.fbx file. Use the following controls to navigate the model;

On Key resume	- Slow Swim
On key press i	- Fast Swim
On key press o	- Dive Out
On key press p	- Dive In
On key press j	- Left Turn
On key press l	- Right Turn
On key press k	- Attack 1
On key press m	- Attack 2
On key press n	- Attack 3
